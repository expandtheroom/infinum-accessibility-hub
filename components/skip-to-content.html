Skip to Content Link

The skip-to-content link is a foundational accessibility pattern that allows keyboard users to bypass repeated navigation.

Design guidelines

UX guidelines

  • The skip link must be fully visible when focused — not just technically focusable. An invisible focused skip link fails WCAG 2.4.7 (Focus Visible).
  • Position it consistently in the top-left of the viewport when it receives focus so keyboard users always know where to look.
  • On complex pages with multiple distinct regions, consider two or three skip links (e.g. "Skip to main content", "Skip to navigation") — but keep the list short to avoid creating its own overhead.

Visual design guidelines

  • The skip link's background color and text must meet the 4.5:1 contrast ratio requirement (WCAG 1.4.3) when it appears on focus.
  • Make the link large enough to read at a glance — a small, low-contrast skip link defeats its own purpose even if it is technically visible.
  • For users who prefer reduced motion, consider displaying the skip link permanently rather than animating it into view, so it is always available without a transition.

Dev guidelines for web

  • Place the skip link as the very first child of <body> — before the <header>, any injected scripts, and cookie banners.
  • The href must point to the main content: <a class="skip-link" href="#main-content">Skip to main content</a>.
  • The target must be programmatically focusable: <main id="main-content" tabindex="-1"> — tabindex="-1" makes it focusable without adding it to the natural tab order.
  • Use the off-screen CSS pattern to hide the link when unfocused: .skip-link { position: absolute; top: -100%; left: 1rem; } .skip-link:focus { top: 0; } — do not use display: none or visibility: hidden, as these remove the link from the accessibility tree.
  • In SPAs with client-side routing, move focus to #main-content on each route change and ensure the skip link remains the first focusable element in the new view.

Example code

<!-- First element inside <body> -->
<a class="skip-link" href="#main-content">
  Skip to main content
</a>

<header role="banner">
  <!-- Site header / navigation -->
</header>

<main id="main-content" tabindex="-1">
  <!-- Main page content -->
</main>
