Sidebar / Drawer Navigation

Accessible sidebar and drawer patterns with focus trapping, keyboard support, and ARIA attributes.

Design guidelines

UX guidelines

  • Include a clearly visible close button (× or "Close") inside the drawer so mouse and touch users can dismiss it without pressing Escape.
  • The overlay drawer must visually block interaction with background content — do not show interactive background elements through the overlay.
  • Opening or closing the drawer must not cause a content jump or layout shift in the main page area.

Visual design guidelines

  • An overlay drawer must include a visible backdrop that dims the background, making it clear the drawer is in the foreground and blocking interaction with page content behind it.
  • All open and close animations must respect prefers-reduced-motion — provide an instant show/hide alternative for users who have reduced motion enabled.
  • A persistent sidebar must be visually distinct from the main content area — use a background color, border, or shadow so users can tell the sidebar apart from the primary content.

Dev guidelines for web

  • The toggle button must use <button aria-expanded="false" aria-controls="drawer-id">; update aria-expanded to "true" when open and "false" when closed.
  • The drawer element must be a <nav aria-label="…" id="drawer-id"> so it is discoverable as a labeled navigation landmark.
  • For a modal overlay drawer: move focus to the first focusable element inside the drawer on open; trap Tab and Shift+Tab within the drawer; pressing Escape closes the drawer and returns focus to the toggle button.
  • Apply inert (preferred) or aria-hidden="true" to the background content container while the drawer is open, so screen reader users cannot accidentally navigate behind it.
  • Remove inert / aria-hidden from background content when the drawer closes.
  • For a persistent sidebar (always visible), no focus trap is needed — it is always in the natural tab order and requires only a correct landmark label.

Example code

<button
  type="button"
  aria-haspopup="dialog"
  aria-expanded="false"
  aria-controls="side-drawer"
  id="drawer-trigger"
>
  Open menu
</button>

<div
  role="dialog"
  id="side-drawer"
  aria-modal="true"
  aria-labelledby="drawer-title"
  hidden
>
  <h2 id="drawer-title">Navigation</h2>
  <nav aria-label="Main navigation">
    <ul>
      <li><a href="/">Home</a></li>
      <li><a href="/about">About</a></li>
      <li><a href="/contact">Contact</a></li>
    </ul>
  </nav>
  <button
    type="button"
    aria-label="Close navigation menu"
    id="drawer-close"
  >
    Close
  </button>
</div>
