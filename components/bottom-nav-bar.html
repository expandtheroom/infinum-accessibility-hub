Bottom Navigation Bar

Accessible bottom navigation bar patterns for mobile and responsive interfaces.

Design guidelines

UX guidelines

  • Every item must pair an icon with a visible text label — never use icons alone without a label visible on screen.
  • Keep the bar to a maximum of 5 items; more than five items crowds the bar and makes labels unreadable on small screens.
  • Maintain consistent item positioning and order across all screens — rearranging items between pages disorients users.
  • When the bar is fixed or sticky, account for its height in page scroll offset so it does not obscure focused content.

Visual design guidelines

  • The active item must be visually distinct using more than color alone — use bold text, an underline, or a persistent indicator bar alongside any color change.
  • Touch targets must be at least 44×44px for each interactive item to meet WCAG 2.5.5 and support switch access users.
  • Ensure sufficient contrast for both icons and labels in all states (default, active, disabled) against the bar's background color.

Dev guidelines for web

  • Wrap the bar in <nav aria-label="Main navigation"> to create a named navigation landmark distinct from other <nav> regions on the page.
  • Mark up items as <ul> with <li><a> for destination links, or <li><button> for in-page actions.
  • Apply aria-current="page" to the active item's <a> or <button> so screen readers announce the current destination.
  • Add aria-hidden="true" to all icon <svg> elements to prevent redundant or meaningless text being added to the accessible name.
  • If an icon-only variant is unavoidable, provide an accessible name via aria-label on the <a> or <button>, or use visually hidden text: <span class="visually-hidden">Home</span>.
  • Icon-only controls without any accessible name are a WCAG 1.1.1 failure and break voice control targeting.

Example code

<nav aria-label="Main navigation">
  <ul role="list">
    <li>
      <a href="/home" aria-current="page">
        <span aria-hidden="true">&#127968;</span>
        <span>Home</span>
      </a>
    </li>
    <li>
      <a href="/search">
        <span aria-hidden="true">&#128269;</span>
        <span>Search</span>
      </a>
    </li>
    <li>
      <a href="/profile">
        <span aria-hidden="true">&#128100;</span>
        <span>Profile</span>
      </a>
    </li>
  </ul>
</nav>
